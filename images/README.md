# Images

This folder is reserved for public-safe visuals.

## Screenshot Rules

Before adding screenshots:

- Use synthetic, mocked, redacted, or explicitly approved data.
- Remove private client names unless approved for public use.
- Remove internal URLs, filenames, buyer identifiers, and raw ticketing details.
- Do not include source code, terminal output with private paths, or build logs.
- Verify charts and tables cannot reveal row-level client data.
